# 异步编程

  • Javascript语言的执行环境是"单线程"。也就是指一次只能完成一件任务。如果有多个任务，就必须排队，前面一个任务完成，再执行后面一个任务
  • 异步任务不具有”堵塞“效应
  • JS 异步编程六种方案 (opens new window)

# React函数组件和类组件的区别

  • React 函数式组件和类组件的区别，不是只有state和性能 (opens new window)
